วิธีการติดตั้งโปรแกรม
1. ติดตั้ง Python 3
2. ติดตั้ง Library จาก requirements.txt
 2.1) เข้าไปที่ cmd (command prompt ผ่าน search)
 2.2) พิมพ์ cd ตามด้วย path ของโฟลเดอร์ build
        Ex. cd Desktop/Stainomic/build 
 2.3) พิมพ์ pip install -r requirements.txt
(สามารถ Double Click ไฟล์ gui.py ใช้งานได้เลย หรือ Open with Python)
